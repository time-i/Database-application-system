此文件目录下存储为前端的页面显示的文件:
	index.html 为最开始的登录界面
	Main.php 为登录成功的显示界面
	student.php为点击学生信息的显示界面
	StudentGradeInfo.php为点击学生成绩后的显示界面
controller文件夹下为实现各种功能的文件：
	包括登录、登出、删除、添加、展示、备份和恢复

css文件下是各种为前端进行页面渲染的文件

js文件下是为前端实现一些功能点文件，例如点击编辑，点击保存

model文件夹下是两个连接数据库的文件
	一个连接时区分用户为教辅或学生，连接不同的mysql的用户（权限不同）
	另一个固定连接root用户
